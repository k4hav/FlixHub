# 🎬 FLIXHUB — Free Entertainment Hub

A stylish animated landing page for sharing free movies, web series, and popular content via Telegram and Google Drive.

## 🌟 Features

- Fully animated gradient background
- Video thumbnail with download buttons
- Google Drive direct download link
- Telegram channel join button
- Responsive design (mobile friendly)
- Pure HTML + CSS (no frameworks)

## 🚀 Live Demo

[Click here to view](https://k4hav.github.io/FlixHub)

## 📁 Tech Stack

- HTML5
- CSS3 (Animations, Gradients)
- Vanilla JavaScript

## 📲 How to Use

1. Clone the repo
2. Open `index.html` in browser
3. Done ✅

## ✏️ Customize

- Change thumbnail image URL in `index.html`
- Update download link and Telegram link
- Edit channel name and description
